# Python

## Basics

To output something to the console in python you use `print()`

```python
print("Hello World")
``` 

### Variables

To define variables in python, you do not need to define the datatype like in java
You can just write:

```python
# Numbers
x = 5
y = 3.14

# String
name = "Max"

# Boolean
is_true = True

# Arrays
fruits = ["Apple", "Orange"]

# List
numbers = {1, 2, 3, 4}
```


### If-Else

Python uses indents instead of `{}` like java

```python
age = 15

if age >= 18:
	print("Adult")
else:
	print("Minor")
``` 


### For-Loop

```python
for fruit in fruits:
	print(fruit)
	
# range
for i in range(5)
	print(i)
``` 


### While-Loop

```python
count = 0

while count < 5:
	print(count)
	count += 1
```
